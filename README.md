# 📚 Curso de Linguagem de Programação

Este repositório contém todas as minhas anotações, resumos e atividades desenvolvidas durante o curso de **Linguagem de Programação**. O material está organizado de forma didática, acompanhando o livro-texto e as listas de exercícios passadas pelo professor.

## 📖 Livro utilizado

> Sebesta, R.W., "Concepts of Programming Languages". Pearson Education, 2019, 12ª edição, 776 páginas.

## 🗂️ Descrição das Pastas

### `livro/`
Contém pastas organizadas por **capítulo do livro** utilizado no curso.  
Cada pasta de capítulo possui um arquivo `resumo_e_atividades.pdf` que inclui:

- Resumo teórico do capítulo
- Atividades do capítulo resolvidas

### `listas_do_professor/`
Contém pastas organizadas por **lista de exercícios** passada pelo professor.  
Cada pasta de lista possui um arquivo `questoes_e_respostas.pdf` que inclui:

- Enunciado das questões
- Respostas desenvolvidas por mim
